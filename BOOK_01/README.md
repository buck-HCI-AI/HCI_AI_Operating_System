## What This Book Is

BOOK_01 is the Hendrickson Construction Operating Manual. It explains how Hendrickson Construction operates using HCI AI — not how to build the system (that is BOOK_00), but how the business runs through the system.

**Core principle:** The software is not the product. The product is a better, repeatable, measurable way to run Hendrickson Construction.

Every project should improve the company.  
Every decision should become knowledge.  
Every KPI should guide action.  
Every workflow should produce evidence.

---

## What BOOK_01 Is Not

- BOOK_01 does not replace SOP 00 or the 42-SOP pack
- BOOK_01 does not replace BOOK_00 (the technical build specification)
- BOOK_01 organizes SOPs, workflows, decisions, KPIs, reports, approvals, and project intelligence into a practical operating manual
